# Netflix Content Analysis 🎬

## Overview
An exploratory data analysis of 8,000+ Netflix titles to uncover content trends, geographic distribution, rating patterns, and year-wise growth using Python and Matplotlib.

## Dashboard Preview
![Netflix Dashboard](netflix_dashboard(1).png)

## Dataset
- 8,000+ Netflix titles
- Source: Kaggle — Netflix Movies and TV Shows Dataset
- Columns: Title, Category, Country, Rating, Release Date

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
1. Movies dominate Netflix content (~70% of total titles)
2. United States produces the most content by far
3. TV-MA is the most common content rating
4. Netflix content grew rapidly between 2015–2019
5. International content increased significantly after 2017


## Why I Built This
I wanted to practice multi-chart dashboard creation on a real-world dataset I could relate to as a Netflix user. The dark theme was intentionally styled to match Netflix branding.

## What I Learned
- Creating 2x2 subplot dashboards in Matplotlib
- Applying custom dark themes to charts
- Handling messy date columns with Pandas
- Visualizing categorical and time-series data together
